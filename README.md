## Orderly — Sorting Algorithm Visualizer

Orderly is an interactive web page that brings sorting algorithms to real-time visualizations. Instead of just reading about how Bubble Sort or Merge Sort works, you can actually watch how the algorithm works at your own speed with elements as bars.

---

## Features

-> 6 sorting algorithms — Bubble, Insertion, Selection, Merge, Quick, and Heap Sort

-> Color-coded bars — orange for comparisons, yellow for swaps, purple for minimum, red for pivot, and green when an element is finally sorted

-> Live comparison and swap counters updating in real time

-> Time complexity and space complexity shown for whichever algorithm you pick

-> Adjustable speed slider — go slow to understand, go fast to watch it fly

-> Custom array input or one-click random array generation

---

##  Tech Stack

-> HTML — structure

-> CSS — styling, animations, light/dark theming using CSS variables

-> JavaScript — all the algorithm logic, race mode

---

## What I Learnt

I understood the difference between algorithms — not just that Merge Sort is faster but why, and what that actually looks like when you watch them race each other.
